REST API
POST /api/orders
Создать новый заказ
{ "customerId": 1, "address": "ул. Ленина 1", "items": [{ "itemId": 10, "quantity": 2 }] }
GET /api/orders/{id}
Получить заказ по ID
POST /api/orders/{id}/pay
Оплатить заказ. Метод CARD — успех, QR — отказ.
{ "paymentMethod": "CARD" }
POST /api/payments internal
Обработка платежа. Вызывается из order-service.
Быстрый старт
1. Запуск инфраструктуры
docker-compose up -d
2. Запуск сервисов
./gradlew :order-service:bootRun --args='--spring.profiles.active=local' ./gradlew :payment-service:bootRun ./gradlew :delivery-service:bootRun --args='--spring.profiles.active=local'
3. Swagger UI
http://localhost:8080/swagger-ui.html # order-service http://localhost:8081/swagger-ui.html # payment-service
